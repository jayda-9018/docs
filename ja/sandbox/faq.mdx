<Warning>
  ### 現在（2025年5月28日以降）正式版のデプロイが完了しました。

  現在の正式API版（`/v1/...`）が**2025年5月28日付でデプロイ完了**しており、新規連携時にはもはやサンドボックスエンドポイントを使用せず、`/v1`**版を連携してください。**

  **すでにサンドボックスに連携して使用中の場合：**\
  2025年6月6日以前に`/sandbox/...` → `/v1/...`にURLのみを修正すれば完了です。

  **サンドボックスを連携して使用したことがない場合：**\
  この文書はスキップして、正式版文書のみを参照すれば十分です。
  

  📄 正式文書を見る：[APIリファレンス](/ja/api-reference/introduction)
</Warning>

<Accordion title="既存のvoice_idが動作しなくなりました。">
  サンドボックスでは、既存の`{character-language-style}`形式のvoice_idは使用できません。\
  現在は**キャラクターベースのvoice_id**のみが使用可能で、スタイルと言語は別のパラメータとして分離されました。

  - `voice_id` → キャラクターベースのVoice ID（例：`91992bbd4758bdcf9c9b0`）
  - `language`, `style` → パラメータとして渡す

  📄 [ボイスIDマッピング表を見る](https://docs.google.com/spreadsheets/d/1RPP9rxE8lWKrkhU91pdHUxOfYGEpqRSDaIybmVBFFZ0/edit?usp=sharing)

  👉 マッピング表にないIDは新バージョンに統合されていない可能性があります。\
  その場合は、カスタマーサポートにお問い合わせください。
</Accordion>

<Accordion title="styleパラメータは必ず入力する必要がありますか？">
  **いいえ、オプションです**\
  styleパラメータはオプションであり、入力しない場合はそのキャラクターに設定されているデフォルトスタイルが自動的に適用されます。

  ただし、キャラクターによって持っているスタイルが異なるため、一部のキャラクターでは **「neutral」以外のスタイルがデフォルト**になっている場合があるため、明確なスタイルが必要な場合は直接指定することをお勧めします。
</Accordion>

<Accordion title="クローンボイスもAPIで使用できますか？">
  **はい、これからは可能です。**\
  Playで作成したクローンボイス（例：'My Voice 1'）もサンドボックスおよび正式APIから呼び出すことができます。
  **ボイスIDを見つける方法：**
  - `/sandbox/voices/search`エンドポイントを呼び出し、
  - `name=My Voice`などの検索語を`query`パラメータに渡すと、
  - 自分のアカウントに登録されているクローンボイスのIDを確認できます。
  ```http
  GET /sandbox/voices/search?name=My Voice
  ```
</Accordion>

<Accordion title="既存のAPI Keyをそのまま使用できますか？">
  **✅ ほとんどのユーザーはそのまま使用可能です。**

  - すでに **コンソールを通じて発行されたAPI Key**を使用している場合、別途措置なく継続して使用できます。

  **⚠️ ただし、次の条件に該当するエンタープライズ顧客は必ず対応が必要です。**

  | 条件                                                | 措置                             |
  | :------------------------------------------------ | :----------------------------- |
  | コンソール登録なしでテキストファイルなどで受け取った **レガシーAPI Key**を使用中の場合 | 必ずコンソールに新規登録後、API Keyを再発行する必要あり |
  | コンソールに登録されているが、 **カスタムボイス設定などが欠落している場合**            | Supertoneと別途協議が必要（ボイス権限移転など）    |

  APIコンソール: https://console.supertoneapi.com
</Accordion>

<Accordion title="サンドボックスでTTS API呼び出し時にクレジットが差し引かれますか？">
  **はい、サンドボックスでもTTS API呼び出し時に実際のクレジットが差し引かれます。**\
  今回のグランドオープンからは**後払いシステムは終了**し、\
  **Playと同じ前払いクレジットベースの料金制**が適用されます。\
  ただし、Predict Duration APIは呼び出し時にクレジットが差し引かれません。

  - クレジット単価はPlayの料金プランと同じです。
  - Playですでにプランを支払っている場合、サンドボックスでも同じクレジットが共有されます。
  - 無料トライアルクレジットも一部提供されます。

  📌 クレジット残高確認API：`GET /sandbox/credits`
</Accordion>

<Accordion title="以前のように後払いを使用することはできませんか？">
  いいえ。**後払いはクローズドベータ専用のポリシー**でした。\
  グランドオープン後はクレジットベースに移行し、**自動決済および未払い使用量精算機能は提供されません。**
</Accordion>

<Accordion title="クローンボイスやカスタムボイスのIDはどのように確認できますか？">
  - 自分のアカウントのすべてのボイス一覧は`/sandbox/voices`または`/sandbox/voices/search`で確認できます。
  - クローンボイスは`"My Voice"`という名前で検索すると確認できます。
  - カスタムボイスがあるエンタープライズアカウントの場合、Supertoneが別途voice_idをお伝えする予定です。
</Accordion>

<Accordion title="styleやlanguageパラメータを間違って入力するとどうなりますか？">
  - 存在しないスタイル/言語を入力すると、**400 Bad Request**エラーが発生します。
  - 必ず`/sandbox/voices`または`/sandbox/voices/search`のレスポンスで提供される値に基づいてパラメータを入力してください。
</Accordion>

<Accordion title="サンドボックスはいつまで使用できますか？">
  サンドボックスは**2025年5月19日〜6月6日**までの運用です。\
  その後は`/sandbox`エンドポイントが終了し、5月28日から`/v1`パスに同じ仕様が正式に反映されるため、v1パスのエンドポイントを使用してください。
</Accordion>

<Accordion title="サンドボックス→v1に戻す際に何を変更する必要がありますか？">
  - API呼び出しパスだけを変更すれば完了です。
    ```vbnet
    /sandbox/text-to-speech → /v1/text-to-speech
    ```
  - ボイスID、パラメータ構造、モデル名などはすべて同じまま維持されます。
</Accordion>
