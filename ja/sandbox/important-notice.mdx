---
title: "サンドボックスとは？"
description: "Supertone APIの構造変更に伴い導入された一時的なテスト環境で、新しい呼び出し方式を事前に確認できます。"
---

Supertone APIは2025年5月28日、クローズドベータ終了と共に公式グランドオープンを迎えます。\
これに伴い、既存API（`v1`エンドポイント）の一部呼び出し構造とパラメータ、ボイスID体系が変更されます。

このような変更点をユーザーの皆様が**無停止で反映してテストできるよう**、\
Supertoneは**2025年5月19日から6月6日まで期間限定のサンドボックス環境**を提供します。

### 🔧 サンドボックスとは？

サンドボックスはSupertone APIの新規呼び出し方式（TTSモデル、voice ID、styleパラメータなど）を**事前に練習できる一時的なテスト環境**です。\
既存の`v1` APIをすでに本番環境に適用しているユーザーも、サンドボックスを通じて新しい構造への移行を事前に試みることができます。

### 🕒 運用期間

> **2025年5月19日（月）～2025年6月6日（金）**\
> 約3週間提供される一時的な環境です。

### 🧭 サンドボックスが必要な理由

- グランドオープンに伴い、既存API呼び出し方式に**重要な構造変更**が生じました。
- 特に`{character-language-style}`形式のボイスIDが**キャラクターID単位に単純化**され、代わりに**styleとlanguageがパラメータとして分離**されました。
- したがって、既存APIコールコードを**すぐに修正するのが難しい**または\
  **サービス中断なしに移行をテストする必要があるユーザー**にサンドボックスが必要です。

### ⚙️ 変更された構造の要約

| 項目 | 既存v1 API | 変更後（サンドボックスおよび正式版） |
| :----- | :----------------- | :------------------------------------ |
| ボイスID | `{serin-ko-happy}` | `{serin}` \+ `language`、`style`パラメータ |
| スタイル指定 | ボイスIDに含まれる | `style`パラメータとして分離 |
| モデル名 | `"pro"`、`"turbo"` | `"sona_speech_1"` |
| テキスト長 | 最大200字 | 最大300字 |
| 課金方式 | 後払い制（1分あたり$0.1） | クレジットベースの料金制（Playと統合） |

### 📝 参考

- サンドボックス環境では**実際にクレジットが差し引かれる**ため、\
  **Playに登録されたプランを通じてクレジットをチャージした後**テストする必要があります。
- 既存の後払い制システムはもはやサポートされておらず、\
  コンソールを通じて発行されていないAPI Keyを使用している場合（エンタープライズ顧客のみ該当）、**APIコンソールを通じてAPI Keyを再発行**する必要がサンドボックス呼び出しが可能になります。
