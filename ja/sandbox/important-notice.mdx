Supertone APIは2025年5月28日、クローズドベータ終了と共に公式グランドオープンを迎えます。\
これにより既存API（`v1`エンドポイント）の一部呼び出し構造とパラメータ、ボイスIDシステムが変更されます。

このような変更事項をユーザーの皆様が**無中断で反映・テストできるよう**、\
Supertoneは**2025年5月19日から6月6日まで限定的なサンドボックス環境**を提供します。

### 🔧 サンドボックスとは？
サンドボックスは、Supertone APIの新規呼び出し方式（TTSモデル、voice ID、styleパラメータなど）を**事前に練習できる一時テスト環境**です。\
既存の`v1` APIをすでにライブ環境に適用しているユーザーも、サンドボックスを通じて新構造への移行を事前に試すことができます。

### 🕒 運営期間
> **2025年5月19日（月）– 2025年6月6日（金）**\
> 約3週間提供される一時環境です。

### 🧭 サンドボックスが必要な理由
- グランドオープンに伴い、既存API呼び出し方式に**重要な構造変更**が生じました。
- 特に`{character-language-style}`形式のボイスIDが**キャラクターID単位に簡素化**され、代わりに**styleとlanguageがパラメータとして分離**されました。
- したがって、既存API呼び出しコードを**すぐに修正することが困難**であったり、\
  **サービス中断なしに移行をテストする必要があるユーザー**にサンドボックスが必要です。

### ⚙️ 変更された構造要約
| 項目     | 既存v1 API          | 変更後（サンドボックス及び正式）                     |
| :----- | :----------------- | :------------------------------------ |
| ボイスID | `{serin-ko-happy}` | `{serin}` + `language`、`style`パラメータ |
| スタイル指定 | ボイスIDに含まれる        | `style`パラメータとして分離                      |
| モデル名    | `"pro"`、`"turbo"` | `"sona_speech_1"`                     |
| テキスト長 | 最大200文字            | 最大300文字                               |
| 課金方式  | 後払い制（1分当たり$0.1）     | クレジットベース料金制（Playと統合）                 |

### 📝 参考
- サンドボックス環境では**実際のクレジットが差し引かれる**ため、\
  **Playに登録されたプランを通じてクレジットをチャージした後**にテストする必要があります。
- 既存の後払い制システムはもはやサポートされず、\
  コンソールを通じて発行されていないAPI Keyを使用している場合（エンタープライズ顧客にのみ該当）、**APIコンソールを通じてAPI Keyを再発行**する必要があり、サンドボックス呼び出しが可能になります。
