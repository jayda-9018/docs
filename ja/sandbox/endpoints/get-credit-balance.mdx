---

このAPIは現在のアカウントの**利用可能なクレジット残高**を照会します。\
Supertone PlayとAPIは同じクレジットシステムを使用しているため、PlayでチャージしたクレジットはAPIにもそのまま適用されます。

### リクエスト方法
```http
GET /sandbox/credits
x-sup-api-key: [YOUR_API_KEY]
```

### レスポンス例
```json
{
  "balance": 146
}
```
→ 現在146クレジットが利用可能であることを意味します。

### 参考事項
- クレジットは**分単位**で課金されます。
- クレジットは**PlayとAPIで共有**されます。
- 新規登録時には基本体験クレジットが提供される場合があります。
