## 시작하기

이 문서는 Supertone API를 처음 사용하시는 분들을 위해 준비된 안내서입니다.  
API를 통해 음성을 생성하고, 보이스를 검색하고, 사용량을 관리하는 전반적인 흐름을 빠르게 익힐 수 있도록 구성되어 있습니다.

Supertone API는 Supertone Play와 함께 통합된 서비스로,  
하나의 계정으로 보이스를 등록하고, 크레딧을 충전하고, 음성 합성까지 이어지는 작업이 모두 가능합니다.  
API 사용에 익숙한 개발자뿐 아니라, Play 중심으로 서비스를 운영하는 사용자에게도 쉽게 접근할 수 있도록 설계되어 있습니다.
