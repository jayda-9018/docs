기존 API(v1)에서 새로운 호출 구조로 전환하기 위해,\
Supertone은 다음과 같은 **2단계 마이그레이션 플로우**를 권장합니다.\
각 단계는 **일정에 맞춰 순차적으로 진행되며**, 샌드박스를 통해 중단 없이 테스트하고 적용할 수 있습니다.

### 1단계: 사전 준비 및 샌드박스 전환 (5월 19일 ~ 5월 27일)

**목표:**\
기존 `v1` API 사용 코드를 `sandbox`**API로 교체하여 테스트** 합니다.

**이 기간에 해야 할 일:**

- 기존 voice_id를 **신규 ID \+ 파라미터(language, style)** 조합으로 교체
- API 호출 URL을 `/v1/...`에서 `/sandbox/...`로 수정
- 기존 레거시 API Key 사용 중인 경우 → 콘솔에서 **신규 API Key 발급**
- 개인 사용자: Play에서 **요금제 가입 후 크레딧 충전**
- 기업 고객: **엔터프라이즈 등록 및 계정/보이스 설정 협의**
- 전체 기능 (TTS, 보이스 검색, 사용량 확인 등) 테스트 진행

> ⏱ _TIP: 이 기간은 실제 서비스 반영 전 최종 점검 기간입니다._

### 2단계: 정식 전환 (5월 28일 ~ 6월 6일)

**목표:**\
정식 그랜드 오픈에 맞춰, API 호출을 다시 `v1`으로 복원합니다.\
`/sandbox/` 경로만 `/v1/`으로 바꾸면 됩니다.

**이후 달라지는 점:**

- `v1` API의 스펙이 sandbox와 동일하게 변경됨
- 크레딧 시스템과 voice_id 구조가 sandbox에서 적용한 방식 그대로 유지됨
- 더 이상 `/sandbox`**는 제공되지 않음** (6월 6일 종료)

### 유의사항

- **레거시 보이스 ID, 모델명, style 없는 호출 등은 5월 28일부터 오류 발생** 가능
- 호출 구조를 sandbox에서 충분히 테스트하지 않은 경우,\
  서비스 중단 위험이 있으니 꼭 사전 테스트를 완료해주세요.
