<Warning>
  ### 안내: 이 문서는 2025년 5월 19일부터 6월 6일까지 제공되는 임시 환경인 **샌드박스(sandbox)**에 대한 문서입니다.

  현재 정식 API 버전`/v1/...`)이 **2025년 5월 28일자로 배포 완료되었으며**, 신규 연동 시에는 **더 이상 샌드박스 엔드포인트를 사용하지 마시고 **`/v1`**버전을 연동해 주세요.** 

  ### 이미 샌드박스에 연동해 사용 중이라면:  

  2025년 6월 6일 이전까지 `/sandbox/...` → `/v1/...`으로 URL만 수정하시면 됩니다.

  ### 샌드박스를 연동해 사용한 적이 없다면:

  이 문서는 건너뛰고, 정식 버전 문서만 참고하셔도 충분합니다.

  📄 정식 문서 보러가기: [API 레퍼런스](/ko/api-reference/introduction)
</Warning>

Supertone API는 2025년 5월 28일, 클로즈드 베타 종료와 함께 공식 그랜드 오픈을 맞이합니다.\
이에 따라 기존 API(`v1` 엔드포인트)의 일부 호출 구조와 파라미터, 보이스 아이디 체계가 변경됩니다.

이러한 변경사항을 사용자 여러분이 **무중단으로 반영하고 테스트할 수 있도록**,\
Supertone은 **2025년 5월 19일부터 6월 6일까지 한시적인 샌드박스 환경**을 제공합니다.

### 🔧 샌드박스란?

샌드박스는 Supertone API의 신규 호출 방식(TTS 모델, voice ID, style 파라미터 등)을 **미리 연습할 수 있는 임시 테스트 환경**입니다.\
기존 `v1` API를 이미 라이브 환경에 적용하고 있는 사용자도, 샌드박스를 통해 새 구조로 미리 마이그레이션을 시도해볼 수 있습니다.

### 🕒 운영 기간

> **2025년 5월 19일(월) – 2025년 6월 6일(금)**\
> 약 3주간 제공되는 임시 환경입니다.

### 🧭 샌드박스가 필요한 이유

- 그랜드 오픈에 따라 기존 API 호출 방식에 **중요한 구조 변경**이 생겼습니다.
- 특히 `{character-language-style}` 형식의 보이스 ID가 **캐릭터 ID 단위로 단순화**되었고, 대신 **style과 language가 파라미터로 분리**되었습니다.
- 따라서 기존 API 호출 코드를 **바로 수정하기 어렵거나**,\
  **서비스 중단 없이 전환을 테스트해야 하는 사용자**에게 샌드박스가 필요합니다.

### ⚙️ 변경된 구조 요약

| 항목     | 기존 v1 API          | 변경 이후 (샌드박스 및 정식)                     |
| :----- | :----------------- | :------------------------------------ |
| 보이스 ID | `{serin-ko-happy}` | `{serin}` \+ `language`, `style` 파라미터 |
| 스타일 지정 | 보이스 ID에 포함됨        | `style` 파라미터로 분리                      |
| 모델명    | `"pro"`, `"turbo"` | `"sona_speech_1"`                     |
| 텍스트 길이 | 최대 200자            | 최대 300자                               |
| 과금 방식  | 후불제 (분당 \$0.1)     | 크레딧 기반 요금제 (Play와 통합)                 |

### 📝 참고

- 샌드박스 환경에서는 **실제 크레딧이 차감**되므로,\
  **Play에 등록된 플랜을 통해 크레딧을 충전한 뒤** 테스트하셔야 합니다.
- 기존 후불제 시스템은 더 이상 지원되지 않으며,\
  콘솔을 통해 발급받지 않은 API Key를 사용하고 있는 경우(엔터프라이즈 고객에만 해당), **API 콘솔을 통해 API Key를 재발급**받아야 샌드박스 호출이 가능합니다.
